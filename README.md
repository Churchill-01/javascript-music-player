A browser-based music player built using vanilla JavaScript (ES6+).
**Features**
Play and pause functionality
Audio state management
Event-driven interaction
Dynamic UI updates

**Technologies Used**
HTML
CSS
JavaScript (ES6+)

**Lessons learnt**
Built a browser-based music player using JavaScript (ES6+).
Implemented play and pause functionality using the HTML Audio API.
Managed audio state with event listeners.
Updated UI dynamically based on playback status.
Structured interactive logic using DOM manipulation.
